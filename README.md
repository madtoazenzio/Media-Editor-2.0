## Variables
````
TG_BOT_TOKEN  - Get bot token from @BOTFATHER

APP_ID        - From my.telegram.org (or @UseTGXBot)

API_HASH      - From my.telegram.org (or @UseTGXBot)

AUTH_USERS : Array to store users who are authorized to use the bot``
````
